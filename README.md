# Student Management System

## About
Console based CLI application in Java to manage student profiles with full CRUD operations.

## Technologies Used
- Java
- HashMap (Collections Framework)
- OOP Principles

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- List All Students
- Input Validation with Exception Handling

## How to Run
1. Clone the repo
2. Open in VS Code
3. Run Main.java
